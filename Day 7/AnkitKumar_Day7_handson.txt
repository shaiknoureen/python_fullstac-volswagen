Service Workers (5 Scenarios)
1.A news website should load the last-read articles even when the user is offline. Which Service Worker features would you use and how?
Ans: We will use the below service workers when the user is offline.
a.	Cache API
b.	Fetch event
c.	IndexedDB (optional for dynamic data)
Now the question is how?
a.	Cache article pages when the user reads them.
b.	Serve cached articles when offline.

2.Your app fetches API data frequently. You want to serve cached data first and update it in the background. Which caching strategy will you choose?
Ans: Caching strategy:
Stale-While-Revalidate
Why:
a.	Fast load from cache
b.	Fresh data fetched in background
How:
a.	Serve cached response immediately
b.	Update cache with network response

3.A Service Worker is registered successfully, but it is not controlling the page. What could be the reasons?
Ans: Possible reasons:
a.	Page not refreshed after registration
b.	Scope mismatch
c.	Service Worker waiting to activate
d.	Page opened before SW registration
Fix:
a.	Reload page
b.	Use clients.claim() in activate event

4.You updated your Service Worker code, but users still see the old cached content. How would you fix this?
Ans: The old cache not deleted.
The solution is:
A . Use cache versioning.
B. Delete old caches in activate.

5.A shopping app must show a custom offline page when the network fails. How would you implement this using a Service Worker?
Ans: How:
a.	Cache an offline.html
b.	Serve it when fetch fails
self.addEventListener("fetch", event => {
  event.respondWith(
    fetch(event.request).catch(() => caches.match("/offline.html"))
  );
});





Styling: Text, Links, Backgrounds & Borders (5 Scenarios)

6.You want all links to:
        A.be blue by default
        B.turn red on hover
        C.remove underline except on hover
How would you write the CSS?
Ans: a {
  color: blue;
  text-decoration: none;
}

a:hover {
  color: red;
  text-decoration: underline;
}

7.A heading text must be centered horizontally and vertically over a background image. Which CSS properties would you use?
Ans: Properties used:
a.	display: flex
b.	justify-content
c.	align-items
.container {
  background-image: url(bg.jpg);
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}

8.You need a gradient background instead of an image to reduce load time. How would you create it using CSS3?
Ans: body {
  background: linear-gradient(to right, #4facfe, #00f2fe);
}

9.A card design requires:
        rounded corners
        a dashed border
        shadow only at the bottom
How would you style it?
Ans:  .card {
  border-radius: 12px;
  border: 2px dashed #ccc;
  box-shadow: 0 6px 6px -3px rgba(0,0,0,0.3);
}

10.A paragraph text must appear justified, with increased line spacing and custom font color. Which properties will you use?
Ans: p {
  text-align: justify;
  line-height: 1.8;
  color: #444;
}

11.You want to animate a button smoothly when hovered without using JavaScript. Which CSS3 features will you use?

Ans: img {
  border-radius: 50%;
  box-shadow: 0 4px 10px rgba(0,0,0,0.3);
}

13.You are building a form and want input fields to change border color smoothly when focused. How can CSS3 transitions help?
Ans: input {
  transition: border-color 0.3s ease;
}

input:focus {
  border-color: blue;
}

14.A designer wants multiple background images on a single div. Is this possible in CSS3? How?
Ans: Yes, possible in CSS3.
div {
  background-image: url(img1.png), url(img2.png);
  background-position: left top, right bottom;
  background-repeat: no-repeat;
}

Box Model & Layout (5 Scenarios)

16.A div with width: 300px becomes larger than expected after adding padding and border. Why does this happen and how do you fix it?
Ans: Padding and border add to width.
Fix:
div {
  box-sizing: border-box;
}

17.Two divs should appear side by side, but they are stacking vertically. What could be the reasons related to box model or layout?
Ans: The reasons are listed below:
Reasons:
a.	display: block
b.	No width set
c.	Parent too narrow
Fix:
div {
  display: inline-block;
}
or use Flexbox:
.parent {
  display: flex;
}


18.A layout breaks on small screens because elements overflow their container. How does box-sizing help?
Ans: It includes padding and border inside width.
* {
  box-sizing: border-box;
}

19.You want equal spacing inside a container without increasing its total width. Which box model concept is important here?
Ans: padding with box-sizing: border-box
.container {
  padding: 20px;
  box-sizing: border-box;
}

20.A developer uses margin-top on a child element, but it affects the parent instead. What box model behavior is this and how can it be avoided?
Ans: Margin collapsing
Avoid by:
a.	Adding padding or border to parent
b.	Using overflow: auto
.parent {
  padding-top: 1px;
}

1. amount = float(input("Enter total shopping amount: "))

if amount >= 5000:
    discount = 0.20
elif amount >= 3000:
    discount = 0.10
else:
    discount = 0

final_amount = amount - (amount * discount)
print("Final payable amount:", final_amount)


2. signal = input("Enter traffic signal color: ").upper()

if signal == "RED":
    print("Stop")
elif signal == "YELLOW":
    print("Get Ready")
elif signal == "GREEN":
    print("Go")
else:
    print("Invalid signal")

9. if marks >= 60:
    print("Pass")
elif marks >= 40:
    print("Conditional Pass")
else:
    print("Fail")

All blocks under if, elif, and else must be properly indented

Incorrect indentation causes IndentationError

Indentation defines which statements belong to which condition
